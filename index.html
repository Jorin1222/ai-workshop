題目二 Lv.2 🟡 進階

表格整理 — 測試格式輸出能力

✅

題目三 Lv.3 🔴 挑戰

文章潤飾 — 測試語感與創意

✅

任務說明

將以下正式公文通知改寫成語氣友善、輕鬆的內部公告，讓同仁更願意閱讀，但須保留所有重要資訊。
難度說明：沒有標準答案——需要你自己判斷哪個版本語感最自然、最符合辦公室文化，最考驗個人判斷力與創意。

原始素材（正式公文）

📋 請複製以下內容，貼入 Prompt 的 [貼上原文] 處

本機關為強化同仁資訊安全意識，訂於本月25日（週三）上午9時至11時，假本機關三樓簡報室辦理『資訊安全教育訓練』課程，請各單位主管轉知所屬同仁，務必準時出席，不得缺席，如有特殊事由無法出席者，應事先以書面向業務承辦人請假。

✏️ 單元二｜Prompt 改寫練習

本單元聚焦在「把模糊 Prompt 改成精準 Prompt」，使用任一 AI 工具操作即可，不需要比較三個工具。

改寫 A

把「請幫我寫信」變成一個好 Prompt

✅

原始（爛）Prompt

問題示範

「請幫我寫信。」

任務說明

這句話缺少哪些資訊？請根據以下情境，把這個 Prompt 補完：

情境：你要寫一封信給廠商，告知因預算縮減，必須延後原訂採購案的執行時程，語氣需禮貌但明確。

先思考「一個好 Prompt 需要哪些要素」，再在下方撰寫你改寫後的版本。

你改寫後的 Prompt（請自己撰寫）

把改寫後的 Prompt 貼到 AI，記錄它的回答

查看參考解答

一個好 Prompt 應該包含：角色設定（你是誰）、任務描述（要做什麼）、背景資訊（為什麼、給誰）、格式要求（語氣、長度）。

參考 Prompt：

你是一名政府採購人員，請幫我撰寫一封正式的公文信件，寄給廠商 XX 公司，說明：本機關因今年度預算調整，原訂 [月份] 執行之採購案需延後至 [新日期] 辦理，請廠商惠予配合。語氣需禮貌、正式，格式符合公務書信規範，字數約 150-200 字。
改寫 B

把「幫我準備會議」變成一個好 Prompt

✅

原始（爛）Prompt

問題示範

「幫我準備會議。」

任務說明

這個 Prompt 太模糊了——AI 不知道什麼會議、準備什麼、給誰看。請根據以下情境補充細節：

情境：你下週要主持一場跨部門協調會議，主題是「推動無紙化作業」，參與者包含資訊部門和各業務組主管，時間 1 小時。

試著讓 AI 不只是列出議程，而是提供一份「開會就能直接用」的完整準備清單。

你改寫後的 Prompt（請自己撰寫）

把改寫後的 Prompt 貼到 AI，記錄它的回答

查看參考解答

要讓 AI 給出實用的回答，關鍵是讓它知道你的角色、會議目的、參與對象和你真正需要的輸出形式。

參考 Prompt：

你是一位資深行政主管，協助我準備一場跨部門協調會議的完整議程。 會議背景：主題為推動無紙化作業、參與者：資訊部門主管及各業務組主管、時間：1 小時。 請提供：(1) 會議開場說明（2 分鐘）、(2) 各議題時間分配建議、(3) 可能遇到的反對意見與建議應對方式。

      ✅
    

    


      
      

        

          
1

          
用 Gemini Canvas 生成 HTML

        


        
開啟 Gemini，在對話框左下角點選 Canvas 模式，然後貼入以下示範 Prompt：


        

          
幫我做一個隨便的儀表板demo頁面+該儀錶板的宣傳頁面，總共要有六種圖表，以深色底為主，資料及主題你幫我訂。最後我要複製到記事本，轉成可以在一般瀏覽器開啟的頁面。

        


        

          
💡 取得 HTML 程式碼的方式

          
Gemini 生成完成後，在 Canvas 畫面右上角點「複製」按鈕，即可複製完整的 HTML 程式碼。

        

      


      
      

        

          
2

          
貼到記事本，存成 .html 檔 ── 大功告成！

        


        

          

            ① 開啟記事本
            Windows 搜尋「記事本」或「Notepad」
          

          

            ② 貼上程式碼
            按 Ctrl+V 貼上複製的 HTML 程式碼
          

          

            ③ 另存新檔
            檔案 → 另存新檔 → 檔名輸入 dashboard.html → 存檔類型選「所有檔案」→ 儲存
          

          

            ④ 開啟！
            找到剛存的 .html 檔案，直接雙擊用瀏覽器開啟 ✅
